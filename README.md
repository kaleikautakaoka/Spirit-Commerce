## Installation
---

Below is the following installing steps to install and run this app.

1. Clone this repo

```sh
git clone 
```

3. Seed the data base

```sh
npm run seed
```

4. Run server

```sh
npm run watch
```

5. Test routes via insomnia

```
Insomnia routes are commented in each route file.
```

6. View study application via :

```
http://localhost:3001
```
